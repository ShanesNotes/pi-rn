# `_derived/`

**Everything in this directory is disposable.**

These files are regenerated from canonical sources by
`scripts/rebuild_derived.py`. Delete them whenever you want; they will come
back on the next run.

**Do not hand-edit** anything here. The validator will warn if it finds a
file without the `generated by` marker.

Canonical sources are:
- `patient.md`
- `constraints.md`
- `timeline/**/encounter_*.md`
- `timeline/**/events.ndjson`
- `timeline/**/vitals.jsonl`
- `timeline/**/notes/*.md`

If a view here disagrees with a canonical source, the canonical source wins
and this file is wrong.
